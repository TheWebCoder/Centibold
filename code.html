The Code for Centibold


                    //get the values from the page
                    //starter or controller function
                    function getValues() {
                        //get values from the page
                        let startValue = document.getElementById("startValue").value;
                        let endValue = document.getElementById("endValue").value;
                        let numbers = [];

                        //We need to validate our input
                        //parse into Integers
                        startValue = parseInt(startValue);
                        endValue = parseInt(endValue);

                        if (Number.isInteger(startValue) && Number.isInteger(endValue)) {
                        //call generateNumbers
                        let numbers = generateNumbers(startValue,endValue);
                        //call displayNumbers
                            displayNumbers(numbers);
                        } else {
                            alert("You must enter integers");
                        }
                    }

                    //generate numbers from start value to the end value
                    //logic function(s)
                    function generateNumbers(sValue, eValue) {

                        let numbers = [];

                        //we want to get all numbers from to start to end

                        for(let index = sValue; index <= eValue; index++) {
                            //this will execute in a loop until index = eValue
                            numbers.push(index)
                        }

                        return numbers;
                    }

                    //display the numbers and mark even numbers bold
                    //display or view function 
                    function displayNumbers(numbers) {

                        let templateRows = "";

                        for (let index = 0; index < numbers.length; index++) {
                            
                            let className = "even";

                            let number = numbers[index];

                            if(number % 2 == 0) {
                                className = "even";
                            }
                            else {
                                className = "odd";
                            }
                            //This does not render correctly with Prism see the source
                            templateRows += `${number}`;
                        }

                        document.getElementById("results").innerHTML = templateRows
                    }

The Code is structured in three functions.

getValues()

This is the start or controller function. It gets triggered via an event listener on the submit button of the app page. In this function we obtain the start value and end value entered by the user. Then after declaring an array, it validates the user input by parsing the values entered as integers. If both the starting value and ending value are valid integers, a call to the generateNumbers function is made and it returns back an array of numbers. However if either value is not a valid integer then an alert is displayed on the browser. Lastly a call to displayNumbers is made by passing in the numbers array as an argument.

generateNumbers(sValue, eValue)

The logic function responsible for generating the numbers based on the two parameters it accepts, the starting value and ending value as integers. After declaring the numbers array all numbers, a for loop is used to populate that array by pushing in the index while it is less than or equal to the ending value. Lastly it returns the numbers as an array.

function displayNumbers(numbers)

Responsible for displaying the numbers and acts as the display or view function. First a template row is declared as an empty string. Next a for loop is used to iterate through all of the numbers of the array. If the number is divisible by 2 it's given a class name of even, otherwise it is given a class name of odd. Lastly the number is added to the template row string which contains the class name in the template literal and that value gets added to the inner HTML of results table.
